The Explorer’s Knapsack

Prescriptive Analytics with Python & PuLP
This repository contains the implementation of a Modified Knapsack Problem using Operations Research (OR) techniques. While a standard knapsack problem focuses only on weight and value, this model introduces Logical Dependencies and Strategic Synergies, mirroring the complexity of real-world decision-making in capital allocation, logistics, and R&D portfolio management.

Project Overview
The goal of this project is to demonstrate how to move from "intuition-based" decisions to Mathematically Guaranteed Optimal Strategies. Using Python and the PuLP library, we solve a scenario where an explorer (or a manager) must select the most valuable items while respecting:

- Physical Constraints: Maximum weight capacity (W).
- Operational Dependencies: Items that require others to function (e.g., a Drone requires a Battery).
- Synergy Bonuses: Extra value generated when specific items are combined (the "1+1=3" effect).
